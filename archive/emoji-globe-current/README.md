# Emoji globe snapshot

Archived before replacing the active pairing surface with the lighter horizontal world-strip picker.

This snapshot keeps the previous canvas globe implementation local in the repository for future reference. It is not loaded by the app and is not deployed unless it is explicitly wired back in.

Files:
- `emoji-globe-current.js` - extracted renderer, inline earth sprite, pointer rotation, swipe-to-room modal hooks.
- `emoji-globe-current.css` - extracted globe shell/canvas/ring styles.
